## What This Project Contains

This repository includes five shell scripts written in bash.
Each script performs a different task related to system inspection and automation.

## Scripts Overview

 1. script1_system_identity.sh

This script displays basic system information such as:

* Kernel version
* Current user
* Uptime
* Date and time
* Operating system



2. script2_package_inspector.sh

This script checks whether Git is installed on the system and displays its version.
It also prints a short description of the software.



 3. script3_disk_auditor.sh

This script audits important directories and shows:

* Permissions
* Owner details
* Disk usage



 4. script4_log_analyzer.sh

This script reads a log file and counts how many times a specific keyword appears.
It helps in identifying patterns like errors in system logs.



 5. script5_manifesto_generator.sh

This script asks the user a few questions and generates a short open-source manifesto in a text file.
